# תיקיית תכנון — משפחת "בית מדרש"

לאחר השוואת 3 כיווני-על, נבחר כיוון **"בית מדרש"** (קלף חם, תכלת #002045 + זהב,
טיפוגרפיית Frank Ruhl Libre, 3 עמודות). התיקייה הזו מכילה את הכיוון הנבחר ועוד
**3 וריאציות בתוך אותו DNA** — אותה פלטה ואותה פרדיגמה, טוויסט שונה.

כל קובץ הוא HTML עצמאי (Tailwind CDN + Google Fonts) — פותחים ישירות בדפדפן, בלי
build ובלי שרת. כולם RTL, עברית, ואותו תוכן לדוגמה (סוגיית **שניים אוחזין**, בבא
מציעא ב׳ ע״א) כדי להשוות תפוח-לתפוח.

## ה-DNA המשותף (קבוע בכל הווריאציות)
- **צבעים:** תכלת `#002045` (ראשי) · זהב `~#7b5800` (אקסנט) · קלף בהיר (רקע).
- **טיפוגרפיה:** Frank Ruhl Libre לתוכן (serif עברי), Heebo ל-UI (sans).
- **פריסה:** בר עליון · דיונים (ימין) · צ'אט (מרכז) · מקורות (שמאל).
- **מוטיב:** מסגרות "טלית" (תכלת/זהב), הבחנה ברורה בין דברי החברותא לשאלות.

## במה כל וריאציה נבדלת

### 1 · קלאסי
המקור. מרקם נייר עדין (grain), בועות צ'אט עם אווטאר ומסגרת טלית, פאנל מקורות
מלא משמאל. **חם, מסורתי, עשיר.** הכי קרוב למה שכבר נבנה ב-React.

### 2 · מודרני
מסיר את המרקם, מוסיף אוויר. כל עמודה היא **כרטיס צף מעוגל** עם צל רך, פינות
פיל (full-round), אקסנט זהב מרוכך. **נקי, עכשווי, נושם** — תחושת מוצר 2025.

### 3 · ספר
האסתטיקה הכי "בית-מדרשית": רקע בעל שוליים מודפסים, **אות פתיחה מעוטרת**
(drop-cap) בתשובות, מפרידים דקורטיביים ✦, וה**מפרשים יושבים בשוליים** כמו בדף
מודפס (רש"י/רמב"ם/תוספות בצבעים שונים). **חגיגי, טקסטואלי, מכובד.**

### 4 · קומפקטי
**עמדת לימוד** למתקדמים: טקסט קטן יותר, יותר שורות במסך, סייד-בר דיונים צפוף,
והמקורות ב**טאבים** (רש"י / גמרא / רמב"ם / תוס') במקום ערימה. **יעיל, צפוף-מידע.**

## מיפוי לקוד הקיים
כל הווריאציות חולקות את אותם נתונים מהשרת — רק שכבת ה-presentation משתנה:

| נתון | היכן מופיע |
|---|---|
| `sessions[]` | סייד-בר ימין (תוכן עניינים בווריאציית "ספר") |
| `messages[]` | בועות / בלוקים עם אות-פתיחה (ספר) |
| `citations[]` | פאנל / כרטיסים / שוליים / טאבים — לפי הווריאציה |
| `intent` | כפתור "מטרה" בקלט |
| `lang` | מתג עב/EN |

הקומפוננטות שיושפעו: `App.tsx`, `SessionSidebar`, `ChatPane`, `MessageBubble`,
`SourcesPanel`/`CitationCard`, `index.css` (הטוקנים זהים בכולן).
